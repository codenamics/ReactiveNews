# ReactiveNews

React app, featching news from NewsAPI. With search functionality, also by range of dates

## Main Technologies

* [x] **JavaScript ES5/ES6**
* [x] **React/Redux**
* [x] **Webpack 4**
* [x] **NewsAPI**
* [x] **SCSS/Bootstrap 4**


### Setup

```bash
$ git clone or download
```

Go to project direction

Install dependencies

```bash
$ npm install

```
Run dev

```bash
$ npm start

```
Run build

```bash
$ npm run build

```
